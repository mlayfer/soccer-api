Instagram Scraper

פרופיל וסטוריז לפי שם משתמש. מתאים ל־RapidAPI: כל משתמש שולח את ה־session שלו ב־Header (לא נשמר בשרת).

Session (אופציונלי — לסטוריז)

לצפייה בסטוריז: העתק את ערך sessionid מהדפדפן (Application → Cookies → instagram.com). נשמר בדפדפן (localStorage) בלבד — לא נשלח לשרת מלבד בבקשות ל־API.

API Info

GET /instagram/info

Profile by Username / פרופיל לפי שם משתמש

GET /instagram/profile/:username

Stories by Username / סטוריז לפי שם משתמש

GET /instagram/stories/:username
הבאת סטוריז פעילים. אם אין תוצאות — הזן Session למעלה (או שלח header X-Instagram-Session).

Stories for Multiple Accounts / סטוריז למספר חשבונות

GET /instagram/stories/:username (multiple)
הזן כמה שמות משתמש מופרדים בפסיק או רווח. לכל משתמש נבצע קריאה ל־/instagram/stories/:username ונציג את כל הסטוריז שלו.

User (Profile + Stories) / משתמש מלא

GET /instagram/user/:username
